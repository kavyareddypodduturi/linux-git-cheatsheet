# Linux & Git Cheat Sheet

## Linux Commands
1. ls - List files in a directory
2. cd - Change directory
3. mkdir - Create a new directory
4. pwd - Show current directory
5. rm - Remove a file or directory

## Git Commands
6. git init - Initialize a new repository
7. git add <file> - Stage file
8. git commit -m "message" - Commit changes
9. git status - Show repository status
10. git branch - List or create branches
11. git checkout <branch> - Switch to another branch
12. git merge <branch> - Merge a branch into current
13. git push - Push commits to GitHub
14. git pull - Pull latest changes from GitHub
15. git clone <repo> - Clone a remote repository
